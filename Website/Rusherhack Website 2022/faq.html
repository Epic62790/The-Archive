FAQ




Q: How do I open the ClickGUI?

A: The default ClickGUI keybind is P. To change this, right click the ClickGUI module and set a bind or type *bind clickgui [key]

Q: How do I type commands?

A: The default command prefix is asterisk (*), to type a command, you must type this prefix before any command you want to type. For a list of commands, type *help

Q: The game is crashing, how do I fix this?

A: Please send a crash log to [email protected] for assistance.

Q: How do I suggest something?

A: Open an issue here.

Q: How do I use Baritone?

A: Download the latest baritone forge api for 1.12.2 here and put it in your Forge mods folder.

Q: Do you plan to support future Minecraft versions?

A: Sometime yes but at the moment 1.12.2 is my priority since that is where most of my target audience is located.
